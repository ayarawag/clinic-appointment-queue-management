Clinic Appointment & Queue Management System


 Project Overview
This system allows patients to register, log in, and book appointments with doctors.
It also supports electronic payments (credit/debit cards and digital wallets).
The system is designed with high cohesion and low coupling to ensure maintainability and scalability



Main Features
 • User Registration: Patients can create new accounts with validation.
 • User Login: Registered patients can log in securely.
 • Book Appointment: Patients can choose a doctor, date, and time slot.
 • Electronic Payment: Supports Visa/MasterCard and digital wallets (PayPal, Google Pay).
 • View Appointments: Display all booked appointments and their status.



 Technologies Used
 • Language: C++
 • Concepts: OOP (Encapsulation, Inheritance, Polymorphism)
 • Tools: File handling, STL vectors, Regex validation, GitHub Projects
