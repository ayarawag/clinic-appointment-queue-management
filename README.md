Clinic Appointment & Queue Management System


Project Overview:
This system allows patients to register, log in, and book appointments with doctors.
It also supports electronic payments (credit/debit cards and digital wallets).
The system is designed with high cohesion and low coupling to ensure maintainability and scalability



Main Features:
 1• User Registration: Patients can create new accounts with validation.
 2• User Login: Registered patients can log in securely.
 3• Book Appointment: Patients can choose a doctor, date, and time slot.
 4• Electronic Payment: Supports Visa/MasterCard and digital wallets (PayPal, Google Pay).
 5• View Appointments: Display all booked appointments and their status.



 Technologies Used
 1• Language: C++
 2• Concepts: OOP (Encapsulation, Inheritance, Polymorphism)
 3• Tools: File handling, STL vectors, Regex validation, GitHub Projects
